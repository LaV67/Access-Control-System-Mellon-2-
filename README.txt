Description:
Access Ñontrol System "Mellon-1" work with two RFID-reader EmMarine 125kHz. 
In this firmware version, the RS485 interface is only used to transfer ID-code of card to the PC.

Instruction:
To add new proxy-card in EEPROM - close jumper SB1 and and attach the card to the reader.Open SB1 for normal mode.
